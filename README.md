# HyRead-eBooks

功能實作項目:

- [x] **實作服務請求**拿到『使用者書單』資料 📕
- [x] **好的架構、乾淨的代碼**
- [x] 有**離線顯示**的功能 
- [x] 有可以**收藏**與**取消收藏**的功能

加分項目實作:
- [x] **創意**來呈現**視覺**或者**使用者體驗**
- [x] **響應式程式設計**
- [ ] 使用者**資料安全保護**
- [x] **技術實作出來**

<br>

> 1. 不確定資料安全保護大概是指什麼實作，或許可以面試的時候再談細節。
